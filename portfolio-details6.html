Microservice based Sterling WMS ARE Billing Module


- Implemented a working prototype of the decoupled Sterling Warehouse Management System Activity Reporting Engine Billing Module using a Spring Boot based Microservice and Cassandra database.
- Created a Docker based application in which the Spring Boot based Microservice communicated with the Cassandra database functioning in separate Docker containers.

Order Management System


- Developed and designed various modules in Fluent Commerce (Order Management System application) for global retailers like Home Delivery, Click and Collect, Subscription and Return Order modules using Core Java, Java 8 features, JUnit and GraphQL.
- Fixed major bugs in the Order Management System application which contributed to increasing the performance of the system by a significant measure and added a revenue of three lacks dollars.
- Mentored a team of multiple application developers to develop new features and handle related issues.
